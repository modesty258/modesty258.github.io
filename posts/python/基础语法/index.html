range 循环#

range(start, stop[, step])
range(stop)

import time
#代表0到10，不包括10
for i in (10):  
    print (i)
time.sleep(3)
#代表1到10，不包括10
for i in (1,10):
    print(i)
#代表1到10，间隔2,不包含10
for i in range(1, 10, 2):
    print(i)

左闭右开#

最左边的start可取，但是右边的step不可取

开始至默认为0#

#开始值不输入的话默认从0开始，打印输出0到9* 
for i in  range(10):    
    print(i)

步长默认为1#

不输入，间隔的时候，间隔为1
